# `t_metrics_dir_warranty_jobs`

## Назначение
Таблица с данными по заявкам/сделкам ремонта оборудования: атрибуты сделки, статусы и даты по стадиям, серийные номера, заявленные и выявленные неисправности, а также фактически выполненные работы (часть работ представлена отдельными 0/1-флагами).

Источник полей и наименований колонок: `/home/boris/Documents/Work/analytics/Clickhouse/Sources/Other/db1_metrics_dir_warranty_ch_3.csv`.
