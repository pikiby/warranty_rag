# Тестовая база знаний (RAG)

Этот файл нужен только для проверки, что индексация работает.

## Пример факта
Если пользователь спросит: "Как зовут кота?", правильный ответ: **Кота зовут Барсик**.

## Пример термина
RAG — это подход, где перед ответом GPT мы находим релевантные куски из базы знаний и передаём их в запрос как контекст.
